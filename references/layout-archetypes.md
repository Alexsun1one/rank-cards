# Layout Archetypes

Choose one archetype by use. A phone thumbnail is ~2cm wide, so covers must survive that:
favor podium / versus / single-hero / pyramid, or a tier-list trimmed to ≤3 rows. Reserve
dense tier-rows and full grids for inside the carousel.

## Tier-row list

Job: the canonical tier list — compare many subjects bucketed into bands.

Layout: horizontal rows, one per tier. Tier label in a colored cap on the LEFT (warm→cool,
top→bottom). Subjects lined up left-to-right inside each row.

Use: carousels and full posts. As a cover only if ≤3 rows and ≤4 per row, else portraits go
tiny.

## Podium / Top-3

Job: crown a clear winner; settle "谁是第一".

Layout: #1 center and tallest, #2 left, #3 right; descending step/bar heights; big medals
🥇🥈🥉.

Use: the best cover archetype — instantly legible, one focal point.

## Pyramid

Job: show scarcity at the top — few elites, many below.

Layout: 1 at the apex, 2–3 in the next band, widening to the base; tier bands as horizontal
slices.

Use: clean cover or hero carousel slide. Ideal for "金字塔" GOAT framing.

## Single-hero card

Job: one subject, FIFA-card style — "X 到底什么段位?"

Layout: one large portrait; big rank/tier badge in a corner; name plate; stat chips down one
side.

Use: strong cover / hook slide. Also: each carousel page = one subject.

## Ranked grid

Job: Top-10 / Top-20 where order matters, equal visual weight.

Layout: N numbered cells, oversized rank numerals, uniform portrait crops in a grid.

Use: carousel. As a cover, only Top-6/9 (3×2 / 3×3) stays readable.

## Head-to-head / versus

Job: a direct 1v1 debate (梅罗, MJ vs LeBron).

Layout: two portraits facing center, split background, a VS glyph, mirrored stat columns.

Use: strong cover — built-in conflict drives clicks.

## Choosing notes

- Crown a winner → podium. Settle a 1v1 → versus. Show scarcity → pyramid. Profile one →
  single-hero. Bucket many → tier-row. Ordered Top-N → grid.
- Cover vs carousel: covers need one focal point; if the layout shrinks to noise at thumbnail
  size, it belongs inside the carousel, not on the cover.
- Never stack two archetypes on one card.
